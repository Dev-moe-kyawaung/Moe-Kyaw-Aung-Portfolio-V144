## 🚀 Quick Start

### Prerequisites
- **Node.js** `>= 18.0.0`
- **npm** `>= 9.0.0`

### 1 — Clone & Install

```bash
git clone https://github.com/Dev-moe-kyawaung/moe-kyaw-aung-portfolio-v144.git
cd moe-kyaw-aung-portfolio-v144
npm install
```

### 2 — Development

```bash
npm run dev
# → http://localhost:5173
```

### 3 — Production Build

```bash
npm run build
# → dist/
```

### 4 — Preview Build Locally

```bash
npm run preview
# → http://localhost:4173
```

## 🌐 GitHub Pages Deploy

### ✅ Automatic — GitHub Actions (Recommended)

Every push to `main` triggers auto-deploy. No manual steps needed after setup.

```bash
# Just push — CI handles the rest
git add .
git commit -m "feat: update portfolio V144"
git push origin main
```

**One-time setup:**
1. Create repo: `moe-kyaw-aung-portfolio-v144`
2. Push all files
3. Go to **Settings → Pages**
4. Source: **GitHub Actions**
5. Wait ~2 minutes → live at:
   `https://Dev-moe-kyawaung.github.io/moe-kyaw-aung-portfolio-v144/`

### Manual Deploy via gh-pages

```bash
npm run build
npm run deploy
```

### ⚠️ Important: Base Path

In `vite.config.js`, the `base` must match your repo name:

```js
// If your repo is: github.com/Dev-moe-kyawaung/moe-kyaw-aung-portfolio-v144
base: '/moe-kyaw-aung-portfolio-v144/'

// If deploying to root (e.g. username.github.io):
base: '/'
```

This portfolio ships as a **Progressive Web App**:

```bash
npm run build && npm run preview
```

## ⚙️ Customization

### Update Personal Info
All content lives in constants at the top of `src/MoeKyawAung_Portfolio_V144.jsx`:

```jsx
const PROFILE = {
  name: "Moe Kyaw Aung",       // ← Your name
  phone1: "+95 9 889 000 889", // ← Phone
  avatar: "https://...",       // ← Profile photo URL
};

const STATS = [
  { value: 12, suffix: "+", label: "Years Exp" },
  // ← Edit numbers
];
```

### Change Color Palette
Edit CSS variables in the `GLOBAL_CSS` string (~line 250):

```css
:root {
  --c-cyan:   #00f0ff;   /* Primary neon — change here */
  --c-pink:   #ff2d78;   /* Secondary neon */
  --c-yellow: #ffe600;   /* Accent */
  --c-bg:     #0a0a0f;   /* Background */
}
```

### Add/Remove Sections
Each section is a self-contained JSX block. Search for section comments:
```jsx
{/* ── PROJECTS ── */}
{/* ── CERTIFICATES ── */}
```

|-------------------|:------------:|
| Performance | 95+ |

**Optimizations applied:**
- `loading="lazy"` on all images (Cloudinary CDN)
- CSS custom properties for O(1) theme switching
- `IntersectionObserver` — animations only fire when in viewport
